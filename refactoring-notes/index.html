Refactoring Lab 03

Refactoring in Software Development

Improving internal structure without changing external behavior.

1. Rename Variables/Methods

Meaningful names improve readability and maintainability. Cryptic names like a, b, x are replaced with descriptive ones.

Before 
public class Calculator {
    public double calc(double a, double b) {
        double x = a + b;
        double y = a * b;
        return x / y;
    }
}
After 
public class Calculator {
    public double calculateSumProductRatio(double num1, double num2) {
        double sum = num1 + num2;
        double product = num1 * num2;
        return sum / product;
    }
}
  • Enhanced readability for future developers.
  • Self-documenting code.
  • Reduces cognitive load when reading logic.

2. Extract Method

Break down long methods into smaller, reusable ones. Each method should do only one thing (Single Responsibility Principle).

Before 
public void printOrderSummary(Order order) {
    double totalPrice = 0;
    for (Item item : order.getItems()) {
        totalPrice += item.getPrice() * item.getQuantity();
    }
    if (order.getCustomer().isMember()) {
        totalPrice *= 0.9;
    }
    // ... printing logic ...
}
After 
public void printOrderSummary(Order order) {
    double totalPrice = calculateTotalPrice(order.getItems());
    totalPrice = applyDiscount(totalPrice, order.getCustomer().isMember());
    printSummary(order, totalPrice);
}

private double applyDiscount(double price, boolean isMember) { ... }
  • Improved modularity and reusability.
  • Easier to unit test individual pieces of logic.
  • Main method acts as a high-level orchestration layer.

3. Guard Clauses

Handle exceptional cases early to reduce nested if statements and "Arrow Code".

Before 
public double calculateDiscount(Order order) {
    double discount = 0.0;
    if (order.getTotalAmount() > 100) {
        if (order.getCustomer().isPremium()) {
            discount = 0.2;
        } else {
            discount = 0.1;
        }
    }
    return discount;
}
After 
public double calculateDiscount(Order order) {
    if (order.getTotalAmount() <= 100) {
        return 0.0;
    }

    if (order.getCustomer().isPremium()) {
        return 0.2;
    }

    return 0.1;
}
